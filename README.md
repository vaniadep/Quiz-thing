# React Quiz App

Проект выполнен по заданию: есть типизация вопроса, массив вопросов, компонент карточки вопроса, страницы Home/Quiz/Result, маршруты между страницами, сохранение лучшего результата в localStorage и стилизация.

## Запуск

```bash
npm install
npm run dev
```

Если папка `node_modules` уже есть, можно сразу запускать:

```bash
npm run dev
```
